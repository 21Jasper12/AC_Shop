## 介紹
---
這是一個電商購物平台。

###  功能
* 輸入寄送地址、運送方式和付款資訊
* 顯示要下訂商品的數量和金額
* 有網頁版和手機板顯示的版面

## 開始使用
---
1. 請先確認有安裝 node.js 與 npm
2. 將專案 clone 到本地
3. 以code方式執行inde.html來顯示網頁
---
## 開發工具
* Node.js 14.16.0
* sass 1.26.7
